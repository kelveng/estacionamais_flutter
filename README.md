# estaciona_mais

Controle de Estacionamento.

## Requisitos

-Dashboards: Vagas disponiveis/Ocupadas, Valor arrecadado, Tabela de preço.
-Relatorio de valor/vaga por dia.
-Mapa do estacionamento.
-Registro de entradas e saídas.

## Requisitos tecnicos

-Cubit
-Dio
-Flutter_Modular
-Clean Architecture
-Testes unitarios
